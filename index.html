Filtro de entidades con control deslizante de tiempo

Este ejemplo demoestro como filtrar datos temporales en una capa GeoJSON usando un widget de control deslizante de tiempo. Esto peude hacerse creando un FeatureFilter y actualizando la propiedad timeExtend del filtro de acuerdo al timeExtend del TimeSlider.

Visualización realista de agua en 3D

Este ememplo muestra como visualizar polígonos de cuerpos de agua con una superficie realista usando WaterSymbol3DLayer. Para lograr una apariencia realista, el usuario puede establecer varias apropiedades, como el color dominante del agua, la fuerza de la ola, la dirección de las olas o seleccionar el tamaño del cuerpo de agua.
